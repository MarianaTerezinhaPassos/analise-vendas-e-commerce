# analise-vendas-e-commerce
7 em cada 10 consumidores brasileiros compraram online mensalmente em 2025 (CNDL/SPC), consolidando o e-commerce como um canal de compras crescente e indispensável. Diante essas informações, este projeto busca identificar gargalos operacionais e oportunidades de crescimento em mercados de alto ticket médio.
